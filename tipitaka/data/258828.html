2.4.1 Anulomapuggala
Mahāvāra
Pariññāvāra
Anulomapuggala
198.
1359 »  Yo kāmarāgānusayaṃ parijānāti so paṭighānusayaṃ parijānātīti? Āmantā.
1360 «  Yo vā pana paṭighānusayaṃ parijānāti so kāmarāgānusayaṃ parijānātīti? Āmantā.
1361 »  Yo kāmarāgānusayaṃ parijānāti so mānānusayaṃ parijānātīti?
1362 Tadekaṭṭhaṃ parijānāti.
1363 «  Yo vā pana mānānusayaṃ parijānāti so kāmarāgānusayaṃ parijānātīti? No.
1364 »  Yo kāmarāgānusayaṃ parijānāti so diṭṭhānusayaṃ…pe…  vicikicchānusayaṃ parijānātīti? No.
1365 «  Yo vā pana vicikicchānusayaṃ parijānāti so kāmarāgānusayaṃ parijānātīti?
1366 Tadekaṭṭhaṃ parijānāti.
1367 »  Yo kāmarāgānusayaṃ parijānāti so bhavarāgānusayaṃ…pe…  avijjānusayaṃ parijānātīti?
1368 Tadekaṭṭhaṃ parijānāti.
1369 «  Yo vā pana avijjānusayaṃ parijānāti so kāmarāgānusayaṃ parijānātīti? No.
199.
1370 »  Yo paṭighānusayaṃ parijānāti so mānānusayaṃ parijānātīti?
1371 Tadekaṭṭhaṃ parijānāti.
1372 «  Yo vā pana mānānusayaṃ parijānāti so paṭighānusayaṃ parijānātīti? No.
1373 »  Yo paṭighānusayaṃ parijānāti so diṭṭhānusayaṃ…pe…  vicikicchānusayaṃ parijānātīti? No.
1374 «  Yo vā pana vicikicchānusayaṃ parijānāti so paṭighānusayaṃ parijānātīti?
1375 Tadekaṭṭhaṃ parijānāti.
1376 »  Yo paṭighānusayaṃ parijānāti so bhavarāgānusayaṃ…pe…  avijjānusayaṃ parijānātīti?
1377 Tadekaṭṭhaṃ parijānāti.
1378 «  Yo vā pana avijjānusayaṃ parijānāti so paṭighānusayaṃ parijānātīti? No.
200.
1379 »  Yo mānānusayaṃ parijānāti so diṭṭhānusayaṃ…pe…  vicikicchānusayaṃ parijānātīti? No.
1380 «  Yo vā pana vicikicchānusayaṃ parijānāti so mānānusayaṃ parijānātīti?
1381 Tadekaṭṭhaṃ parijānāti.
1382 »  Yo mānānusayaṃ parijānāti so bhavarāgānusayaṃ…pe…  avijjānusayaṃ parijānātīti? Āmantā.
1383 «  Yo vā pana avijjānusayaṃ parijānāti so mānānusayaṃ parijānātīti? Āmantā.
201.
1384 »  Yo diṭṭhānusayaṃ parijānāti so vicikicchānusayaṃ parijānātīti? Āmantā.
1385 «  Yo vā pana vicikicchānusayaṃ parijānāti so diṭṭhānusayaṃ parijānātīti? Āmantā…pe… .
202.
1386 »  Yo vicikicchānusayaṃ parijānāti so bhavarāgānusayaṃ…pe…  avijjānusayaṃ parijānātīti?
1387 Tadekaṭṭhaṃ parijānāti.
1388 «  Yo vā pana avijjānusayaṃ parijānāti so vicikicchānusayaṃ parijānātīti? No.
203.
1389 »  Yo bhavarāgānusayaṃ parijānāti so avijjānusayaṃ parijānātīti? Āmantā.
1390 «  Yo vā pana avijjānusayaṃ parijānāti so bhavarāgānusayaṃ parijānātīti? Āmantā. (Ekamūlakaṃ.)
204.
1391 »  Yo kāmarāgānusayañca paṭighānusayañca parijānāti so mānānusayaṃ parijānātīti?
1392 Tadekaṭṭhaṃ parijānāti.
1393 «  Yo vā pana mānānusayaṃ parijānāti so kāmarāgānusayañca paṭighānusayañca parijānātīti? No.
1394 »  Yo kāmarāgānusayañca paṭighānusayañca parijānāti so diṭṭhānusayaṃ…pe…  vicikicchānusayaṃ parijānātīti? No.
1395 «  Yo vā pana vicikicchānusayaṃ parijānāti so kāmarāgānusayañca paṭighānusayañca parijānātīti?
1396 Tadekaṭṭhaṃ parijānāti.
1397 »  Yo kāmarāgānusayañca paṭighānusayañca parijānāti so bhavarāgānusayaṃ…pe…  avijjānusayaṃ parijānātīti?
1398 Tadekaṭṭhaṃ parijānāti.
1399 «  Yo vā pana avijjānusayaṃ parijānāti so kāmarāgānusayañca paṭighānusayañca parijānātīti? No. (Dukamūlakaṃ.)
205.
1400 »  Yo kāmarāgānusayañca paṭighānusayañca mānānusayañca parijānāti so diṭṭhānusayaṃ…pe…  vicikicchānusayaṃ parijānātīti? Natthi.
1401 «  Yo vā pana vicikicchānusayaṃ parijānāti so kāmarāgānusayañca paṭighānusayañca mānānusayañca parijānātīti?
1402 Tadekaṭṭhaṃ parijānāti.
1403 »  Yo kāmarāgānusayañca paṭighānusayañca mānānusayañca parijānāti so bhavarāgānusayaṃ…pe…  avijjānusayaṃ parijānātīti? Natthi.
1404 «  Yo vā pana avijjānusayaṃ parijānāti so kāmarāgānusayañca paṭighānusayañca mānānusayañca parijānātīti?
1405 Mānānusayaṃ parijānāti. (Tikamūlakaṃ.)
206.
1406 »  Yo kāmarāgānusayañca paṭighānusayañca mānānusayañca diṭṭhānusayañca parijānāti so vicikicchānusayaṃ parijānātīti? Natthi.
1407 «  Yo vā pana vicikicchānusayaṃ parijānāti so kāmarāgānusayañca paṭighānusayañca mānānusayañca diṭṭhānusayañca parijānātīti?
1408 Diṭṭhānusayaṃ parijānāti kāmarāgānusayañca paṭighānusayañca mānānusayañca tadekaṭṭhaṃ parijānāti…pe… . (Catukkamūlakaṃ.)
207.
1409 »  Yo kāmarāgānusayañca paṭighānusayañca mānānusayañca diṭṭhānusayañca vicikicchānusayañca parijānāti so bhavarāgānusayaṃ…pe…  avijjānusayaṃ parijānātīti? Natthi.
1410 «  Yo vā pana avijjānusayaṃ parijānāti so kāmarāgānusayañca paṭighānusayañca mānānusayañca diṭṭhānusayañca vicikicchānusayañca parijānātīti?
1411 Mānānusayaṃ parijānāti. (Pañcakamūlakaṃ.)
208.
1412 »  Yo kāmarāgānusayañca paṭighānusayañca mānānusayañca diṭṭhānusayañca vicikicchānusayañca bhavarāgānusayañca parijānāti so avijjānusayaṃ parijānātīti? Natthi.
1413 «  Yo vā pana avijjānusayaṃ parijānāti so kāmarāgānusayañca paṭighānusayañca mānānusayañca diṭṭhānusayañca vicikicchānusayañca bhavarāgānusayañca parijānātīti?
1414 Mānānusayañca bhavarāgānusayañca parijānāti. (Chakkamūlakaṃ.)
